Três formas de encontrar
o centro dos dados
Quando trabalhamos com variáveis quantitativas, utilizamos as MTC para expressar, através de um único número, em torno de que valor tende a se concentrar um conjunto de dados.
X̄

Média

A MTC mais utilizada e melhor compreendida. Calcula-se pelo somatório de todos os valores dividido pelo número de observações.
Média = Σ valores ÷ n
Exemplo: (0,6 + 0,8 + ... + 0,7) ÷ 20 = 0,8
✓ Considera todos os valores
✗ Influenciada por extremos
✓ Usada em testes estatísticos
✗ Só para distribuição simétrica
✓ Facilmente compreendida
M̃

Mediana

Ordenando os dados, é o valor que divide o conjunto em duas partes iguais. Se n for par, é a média dos dois valores centrais.
n ímpar → valor central
n par → (valor₁ + valor₂) ÷ 2
20 idosos: (0,8 + 0,8) ÷ 2 = 0,8
Nota: O artigo original (Exemplo 2, p. 192) apresenta o resultado 0,75, porém a sequência ordenada exibida contém 21 valores em vez dos 20 da Tabela 1 (um valor 0,8 a mais). Recalculando com os dados corretos da Tabela 1, as posições 10ª e 11ª são ambas 0,8, resultando em mediana = 0,8.
✓ Não sofre influência de extremos
✗ Pouco compreendida
✓ Funciona em dados assimétricos
✗ Pouco usada em testes
✓ Também funciona em simétricos
Mo

Moda

O valor que mais se repete em um conjunto de dados. Sua utilização é menos frequente, mas demonstra qual valor é o mais comum.
Moda = valor mais frequente
Idades da turma: 7 aparece 8 vezes → Moda = 7
✓ Simples de identificar
✗ Uso pouco frequente
✓ Mostra o valor mais comum
✗ Pode não existir

Quando usar cada medida? Vantagens, desvantagens e situações de aplicação.
MTC Quando usar Vantagens Desvantagens
Média (X̄) Distribuição simétrica (Gaussiana) Considera todos os valores; usada em testes estatísticos; facilmente compreendida Influenciada por valores extremos; inadequada para distribuições assimétricas
Mediana (Md) Distribuição assimétrica (e também simétrica) Não sofre influência de extremos; versátil para qualquer distribuição Propriedades pouco compreendidas; pouco usada em testes estatísticos
Moda (Mo) Quando se quer o valor mais frequente Simples de identificar; mostra o valor mais comum do conjunto Uso pouco frequente; pode não existir se todos os valores forem diferentes
01

Distribuição simétrica

Média = Mediana = Moda. Os dados se distribuem igualmente para ambos os lados, formando o formato de sino. Qualquer MTC pode representar o centro.

02

Distribuição assimétrica

Média ≠ Mediana. Valores extremos "puxam" a média. Use preferencialmente a mediana, pois ela não é influenciada por extremos.

03

Importância clínica

Avaliar se as MTC foram corretamente aplicadas é fundamental, pois conclusões baseadas em MTC inadequada podem estar distorcidas.
