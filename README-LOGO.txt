====================================================
  INSIRA A LOGO DA FACULDADE AFYA GUANAMBI AQUI
====================================================

Salve o arquivo da logo com o nome:

    logo-afya.png

Nesta mesma pasta  (public/)

Formatos aceitos:  PNG (recomendado) ou SVG
Tamanho ideal:     altura de 80–120 px (ou versão @2x para telas Retina)

A logo aparecerá automaticamente:
  • no Hero (tela inicial)
  • no Encerramento (tela final)

Após adicionar a logo, reconstrua o projeto:
    npm run build
